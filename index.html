Number Plate Recognition Project

About the Project

This project is designed to detect and extract vehicle registration numbers from images using artificial intelligence. It leverages the power of the Google Vision API for accurate and efficient number plate recognition.

Features

  • Upload images of cars directly.
  • Extracts text from vehicle registration number plates using AI.
  • High-speed and accurate processing.

How It Works

The system processes images through the Google Vision API, which detects text in the number plate area. The extracted text is displayed on-screen for easy use.
